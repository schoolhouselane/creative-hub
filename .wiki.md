# Project Summary
The SHL Brief System is an AI-driven creative agency platform designed to streamline the process of submitting brand briefs and generating marketing assets. By integrating advanced AI tools, this platform enhances the creative workflow for designers, enabling the efficient production of diverse marketing materials such as logos, advertisements, social media content, and videos, thus improving brand identity and marketing strategies.

# Project Module Description
The project consists of several functional modules:
- **Frontend**: User interface for submitting briefs, selecting AI tools, and managing projects.
- **Backend**: API services for handling user authentication, brief submissions, and AI tool integrations.
- **Database**: Stores user data, briefs, and generated assets.

# File Description Inventory
- **app/backend/core/**: Contains core functionalities like authentication and database configurations.
- **app/backend/models/**: Defines data models for briefs and brand profiles.
- **app/backend/routers/**: API endpoints for interacting with the application.
- **app/frontend/src/**: Contains all frontend source files, including components, pages, and styles.
- **app/frontend/src/pages/**: Updated to include new pages for managing briefs and integrating a sidebar layout.

# Technology Stack
- **Frontend**: React, TypeScript, Vite, Tailwind CSS
- **Backend**: FastAPI, Python, SQLAlchemy
- **Database**: PostgreSQL
- **AI Tools**: Various AI models for text, image, and video generation

# Usage
1. **Install Dependencies**:
   - For the backend: Navigate to `app/backend` and run `pip install -r requirements.txt`.
   - For the frontend: Navigate to `app/frontend` and run `pnpm install`.

2. **Build the Project**:
   - For the backend: Run `uvicorn main:app --reload` to start the backend server.
   - For the frontend: Run `pnpm run build` to build the frontend assets.

3. **Run the Application**:
   - Start both backend and frontend servers according to the instructions provided in their respective directories.
